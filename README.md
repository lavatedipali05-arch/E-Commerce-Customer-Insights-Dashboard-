## 📊 Project Overview
This project presents an interactive E-commerce Customer Insights Dashboard created using Power BI.  
It helps analyze customer behavior, income, time spent, and location-based insights.

---

## 🛠️ Tools Used
- Microsoft Power BI
- Excel / CSV Dataset

---

## 📌 Features
- Total Customers Summary
- Gender Distribution (Pie Chart)
- Annual Income Analysis (Gauge Chart)
- Customer Data by Location (Table)
- Total Time Spent Analysis (Area Chart)
- Location-wise Customer Distribution (Bar Chart)
- Interactive Filters (Gender, Location, etc.)
